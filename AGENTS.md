# AI Coding Agents Guide

This document provides guidance on working with various AI coding agents in this repository.

## Overview

AI coding agents can significantly enhance development productivity when used effectively. This guide covers best practices for working with different AI assistants.

## Available Agents

### GitHub Copilot
- Integrated directly into supported IDEs
- Provides real-time code suggestions
- Excellent for completing repetitive patterns
- Best used with clear, descriptive comments

### Claude (Anthropic)
- Strong reasoning capabilities
- Good for architectural decisions
- Excellent at explaining complex code
- See CLAUDE.md for specific instructions

### Gemini (Google)
- Multi-modal capabilities
- Good for documentation and analysis
- Strong at code review and suggestions
- See GEMINI.md for specific instructions

## Best Practices

### Writing Effective Prompts
1. **Be Specific**: Clearly describe what you want the AI to do
2. **Provide Context**: Include relevant background information
3. **Set Constraints**: Specify any limitations or requirements
4. **Request Examples**: Ask for code examples when helpful

### Code Review with AI
- Use AI to review code for potential issues
- Ask for suggestions on optimization
- Get explanations of complex algorithms
- Request testing strategies

### Documentation
- Generate initial documentation drafts
- Create API documentation
- Write user guides and tutorials
- Update existing documentation

## Team Collaboration

### Consistency
- Establish team conventions for AI usage
- Share effective prompts and patterns
- Review AI-generated code together
- Maintain coding standards

### Quality Assurance
- Always review AI-generated code
- Test thoroughly before merging
- Validate security implications
- Ensure compliance with project standards

## Troubleshooting

If an AI agent provides unexpected results:
1. Refine your prompt with more specific details
2. Break complex requests into smaller parts
3. Provide more context about the codebase
4. Ask for alternative approaches

## Getting Help

For specific questions about individual agents, refer to their respective documentation files in this repository.
