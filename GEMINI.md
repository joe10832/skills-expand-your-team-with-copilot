# Gemini AI Assistant Instructions

Gemini is Google's multimodal AI model that excels at understanding and generating various types of content including code, documentation, and visual elements.

## Strengths

- **Multi-modal Capabilities**: Can work with code, images, documents, and data
- **Code Generation**: Strong at generating code in multiple programming languages
- **Data Analysis**: Excellent at analyzing and explaining data patterns
- **Documentation**: Great at creating user-friendly documentation
- **Integration Planning**: Good at suggesting integrations with Google Cloud services

## Best Practices for Working with Gemini

### Code Development

#### Web Development
```
Example prompt:
"Create a responsive React component for a product card that displays product image, title, price, and description. Include proper TypeScript types and CSS styling that works well on mobile devices."
```

#### Data Processing
```
Example prompt:
"Write a Python function that processes a CSV file containing sales data, calculates monthly totals, and generates a summary report. Include error handling for malformed data."
```

#### API Integration
```
Example prompt:
"Help me integrate with the Google Maps API to display store locations on a web page. Include proper error handling and loading states."
```

### Documentation and Analysis

#### API Documentation
- Generate OpenAPI/Swagger specifications
- Create user guides with examples
- Write integration tutorials

#### Code Analysis
- Review code for best practices
- Suggest performance improvements
- Identify potential security issues

### Multi-modal Features

#### Working with Images
- Analyze UI mockups and generate corresponding code
- Process screenshots for bug reporting
- Generate code from hand-drawn diagrams

#### Data Visualization
- Create charts and graphs from datasets
- Generate visualization code
- Analyze data patterns and trends

## Integration Strategies

### With GitHub Copilot
- Use Gemini for initial code structure and architecture
- Use GitHub Copilot for detailed implementation
- Have Gemini review and optimize the final code

### With Google Cloud Platform
- Leverage Gemini's knowledge of GCP services
- Get help with cloud architecture decisions
- Generate deployment configurations

### Development Workflow
1. Use Gemini to understand requirements and plan approach
2. Generate initial code structure
3. Implement details with GitHub Copilot
4. Use Gemini to create documentation and tests
5. Have Gemini review for optimization opportunities

## Code Quality Guidelines

### Best Practices Requests
- Always ask for proper error handling
- Request TypeScript types for JavaScript code
- Ask for unit tests alongside implementations
- Request performance considerations

### Example Quality-Focused Prompt
```
"Generate a Python class for managing user sessions with Redis backend. Include proper error handling, type hints, comprehensive docstrings, and unit tests. Also suggest performance optimizations and security considerations."
```

## Tips for Optimal Results

1. **Be Comprehensive**: Provide full context including tech stack and requirements
2. **Request Examples**: Ask for concrete code examples rather than abstract concepts
3. **Specify Constraints**: Mention performance, security, or compatibility requirements
4. **Iterate and Refine**: Build on responses to get more detailed implementations
5. **Validate Outputs**: Always test generated code and verify suggestions
6. **Combine Capabilities**: Leverage Gemini's multi-modal nature for richer interactions
