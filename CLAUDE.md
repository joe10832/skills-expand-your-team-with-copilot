# Claude AI Assistant Instructions

Claude is an AI assistant created by Anthropic that excels at reasoning, analysis, and detailed explanations.

## Strengths

- **Code Analysis**: Excellent at understanding complex codebases and explaining how they work
- **Architecture Design**: Strong at suggesting system architecture and design patterns
- **Documentation**: Great at writing clear, comprehensive documentation
- **Code Review**: Thorough in identifying potential issues and improvements
- **Debugging**: Effective at helping diagnose and fix complex bugs

## Best Practices for Working with Claude

### Effective Prompting
- Provide comprehensive context about your project
- Be specific about your requirements and constraints
- Ask follow-up questions to clarify suggestions
- Request explanations for complex recommendations

### Code Development
```
Example prompt:
"I'm building a REST API in Python using FastAPI. I need to implement user authentication with JWT tokens. Can you help me design the authentication flow and provide example code that follows security best practices?"
```

### Code Review
```
Example prompt:
"Please review this Python function for potential security vulnerabilities, performance issues, and adherence to best practices. Also suggest any improvements for readability and maintainability."
```

### Architecture Planning
```
Example prompt:
"I'm designing a microservices architecture for an e-commerce platform. The system needs to handle user management, product catalog, orders, and payments. Can you suggest a service breakdown and communication patterns?"
```

## Integration Tips

### With GitHub Copilot
- Use Claude for high-level planning and architecture decisions
- Use GitHub Copilot for implementing the detailed code
- Have Claude review what Copilot generates

### Documentation Workflow
1. Write code with GitHub Copilot
2. Use Claude to generate comprehensive documentation
3. Have Claude explain complex algorithms or design decisions
4. Use Claude to create user guides and API documentation

### Problem-Solving Process
1. Describe the problem to Claude in detail
2. Ask for multiple solution approaches
3. Discuss trade-offs and implications
4. Get help breaking down complex solutions into implementable steps

## Code Quality Guidelines

When working with Claude on code:
- Always ask for tests along with implementation code
- Request error handling and edge case considerations
- Ask for performance implications of suggested solutions
- Have Claude explain any complex algorithms or patterns

### Legacy Code Modernization
- Understanding old codebases

### Performance Optimization
- Identifying bottlenecks
- Suggesting optimization strategies

### Security Review
- Identifying potential vulnerabilities

## Tips for Better Results

1. **Provide Context**: Share relevant code, requirements, and constraints
2. **Iterate**: Build on previous responses and ask follow-up questions
3. **Be Specific**: Ask for concrete examples rather than general advice
4. **Validate**: Always test and validate Claude's suggestions
5. **Combine Strengths**: Use Claude's analysis with other tools' implementation capabilities
